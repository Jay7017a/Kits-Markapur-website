🏫 Krishna Chaithanya Institute of Technology and Sciences – College Website 📌 Project Overview

This is a front-end college website project developed for Krishnachetna Institute of Technology and Sciences. The project is built completely using HTML, CSS, and Bootstrap without any backend implementation. It focuses on creating a responsive, structured, and user-friendly interface for college-related information.

⚙️ Technologies Used HTML5 CSS3 Bootstrap 5 JavaScript (basic Bootstrap interactions) 📁 Project Directory Structure /project-root │
├── image/ # All images used in website
├── home.html # Home page ├── alumni.html # Alumni page
├── contact.html # Contact page
├── gallery.html # Gallery page 
└── README.md

🏠 Home Page (home.html) Features

The home page is the main section of the website and includes multiple interactive and informational components:

🔔 Flash Message Section – Displays important announcements in a scrolling/transition style format 📢 College Announcement Board – Dynamic transition-based notice display 🎠 Image Carousel – Slideshow of college-related images with smooth transitions 🏛️ Examination Board & Notification Section – Displays exam updates and important notices 👤 About Secretary Section – Information with corresponding images 🏢 Placement Partners Section – Showcases companies associated with placements 🎓 Recently Placed Students Section – Displays student placement details and company names 📞 Contact Information Section – Basic contact details for communication 📄 Other Pages 🎓 Alumni Page (alumni.html) Displays alumni-related information and details of passed-out students. 📷 Gallery Page (gallery.html) Contains images showcasing college events, activities, and campus life. 📞 Contact Page (contact.html) Includes contact details of college administration and inquiry form-style layout. Principal and staff contact emails are provided for queries and communication. 📸 Screenshots

Screenshots of the home page and other sections have been included in the repository to demonstrate UI layout and design structure.

📊 Project Result

This project successfully demonstrates a responsive and structured front-end college website using Bootstrap components. It focuses on layout design, responsiveness, and UI presentation without backend integration.
